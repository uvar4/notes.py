<h1>Notes.py</h1>

A small notes-like project i made in python in about 10 minutes. One of the stupidest project you'll ever see on GitHub, but whatever, i just felt like putting it here.

<h1>How to use: </h1>

python notes.py --add_your_command_here NAME_HERE

Oh yeah, talking about commands, there's only one: --add. It will add stuff you ask it to in a notes.txt file.
